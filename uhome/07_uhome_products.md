# uHOME Product Split
uHOME-client = kiosk, controller, local-network UI surfaces
uHOME-server = Linux-based Steam and local-network host runtime, vault reader,
Beacon Activate content surfaces, persistent services
uHOME-empire = always-on Google or HubSpot sync, webhook-style server actions,
console CRM and workflow UI, HubSpot sync for app-owned contact records
